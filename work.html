Safe Travels - HTML, SCSS, JS, API (Rails framework)

For my final project at Le Wagon, Berlin I pitched the idea of a safe travel app for women. As a woman that travels a lot I wanted to create an app that utilizes APIs with safety ratings to plan a trip to a new city.

The website uses governmental ratings, user reviews, and recommendations to help women plan trips all in one app, and build their own trip plan.

I was lucky enough to be the project manager for a great team of 5. We started by creating a Figma design, and then proceeding to build the project on Rails. I focused on front-end during the project, in particular HTML, SCSS, and JS(Stimulus). As well as APIs, and seed creation.
